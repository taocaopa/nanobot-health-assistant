# 健康提醒技能

## 功能

- 设置用药提醒（时间、剂量、频次）
- 喝水提醒
- 运动提醒
- 眼保健操/休息提醒
- 睡眠时间提醒

## 数据结构

提醒配置存储在 `~/.nanobot/data/health_reminders.json`:

```json
{
  "reminders": [
    {
      "id": "rem_001",
      "name": "维生素D",
      "type": "medication",
      "schedule": "daily",
      "time": "08:00",
      "dosage": "1片",
      "notes": "早餐后服用",
      "enabled": true
    },
    {
      "id": "rem_002",
      "name": "喝水提醒",
      "type": "water",
      "schedule": "interval",
      "interval_minutes": 60,
      "enabled": true
    },
    {
      "id": "rem_003",
      "name": "眼保健操",
      "type": "break",
      "schedule": "interval",
      "interval_minutes": 120,
      "enabled": true
    }
  ]
}
```

## 使用方法

### 设置用药提醒

- "每天8点提醒吃维生素D，1片"
- "设置早晚两次的降压药提醒"
- "每周一、三、五提醒补钙"

### 设置喝水提醒

- "每小时提醒喝水"
- "设置每天8杯水的提醒"

### 设置休息提醒

- "每2小时提醒休息眼睛"
- "设置久坐提醒，每小时站起来"
- "晚上10点提醒睡觉"

### 管理提醒

- "查看所有提醒"
- "暂停喝水提醒"
- "删除维生素提醒"

## 提醒类型

| 类型 | 用途 | 示例 |
|------|------|------|
| medication | 用药提醒 | 维生素、处方药 |
| water | 喝水提醒 | 每小时一杯水 |
| exercise | 运动提醒 | 下午6点去跑步 |
| break | 休息提醒 | 眼保健操、站立 |
| sleep | 睡眠提醒 | 就寝时间 |
| meal | 用餐提醒 | 按时吃饭 |

## 提醒频率

- **daily**: 每天固定时间
- **weekly**: 每周特定几天
- **interval**: 每隔一段时间
- **custom**: 自定义 Cron 表达式

## 注意事项

1. **用药提醒**：务必核对药品名称和剂量，错误用药有害健康
2. **处方药**：请遵医嘱，不要自行调整用药时间和剂量
3. **提醒延迟**：由于技术原因，提醒可能有1-2分钟延迟
4. **紧急情况**：健康提醒不能替代紧急医疗救助

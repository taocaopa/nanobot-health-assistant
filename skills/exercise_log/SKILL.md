# 运动记录技能

## 功能

- 记录各类运动活动
- 计算卡路里消耗
- 追踪运动目标和进度
- 生成运动报告

## 数据结构

运动记录存储在 `~/.nanobot/data/exercise_logs.json`:

```json
{
  "2026-03-15": {
    "activities": [
      {
        "type": "running",
        "name": "慢跑",
        "duration": 30,
        "unit": "minutes",
        "distance": 5,
        "distance_unit": "km",
        "calories": 300,
        "intensity": "moderate",
        "time": "07:00",
        "notes": "晨跑，感觉很棒"
      }
    ],
    "total_duration": 30,
    "total_calories": 300
  }
}
```

## 使用方法

### 记录运动

- "跑步 30 分钟，5公里"
- "记录游泳 45 分钟"
- "做了 20 分钟瑜伽"
- "步行 10000 步"

### 查询记录

- "这周运动了几次？"
- "显示本月运动统计"
- "本周消耗了多少卡路里？"

### 运动目标

- "设定每周运动 5 次的目标"
- "查看运动目标进度"
- "建议今天的运动"

## 常见运动消耗（60kg成人）

| 运动 | 30分钟消耗 | 强度 |
|------|-----------|------|
| 慢跑 | 300 kcal | 中等 |
| 快走 | 150 kcal | 低 |
| 游泳 | 250 kcal | 中等 |
| 骑行 | 200 kcal | 中等 |
| 瑜伽 | 120 kcal | 低 |
| 力量训练 | 180 kcal | 中高 |
| 跳绳 | 350 kcal | 高 |

## 运动建议

### WHO 推荐标准
- **成年人**: 每周至少 150-300 分钟中等强度运动
- **或**: 75-150 分钟高强度运动
- **肌肉训练**: 每周至少 2 次

### 运动类型搭配
1. **有氧运动**: 跑步、游泳、骑行（心肺功能）
2. **力量训练**: 举重、俯卧撑、深蹲（肌肉骨骼）
3. **柔韧性**: 瑜伽、拉伸（关节灵活）

## 注意事项

1. 运动前热身，运动后拉伸
2. 循序渐进，避免过度训练
3. 如有不适立即停止，必要时就医
4. 慢性病患者运动前咨询医生

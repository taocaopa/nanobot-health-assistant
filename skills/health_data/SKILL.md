# 健康数据管理技能

## 功能

- 记录各类健康指标（体重、血压、心率、血糖、睡眠时长等）
- 查询历史数据
- 生成趋势分析
- 提供健康建议

## 数据结构

健康指标存储在 `~/.nanobot/data/health_metrics.json`:

```json
{
  "weight": [
    {"date": "2026-03-15", "value": 70.5, "unit": "kg", "note": "早餐后"}
  ],
  "blood_pressure": [
    {"date": "2026-03-15", "systolic": 120, "diastolic": 80, "unit": "mmHg"}
  ],
  "heart_rate": [
    {"date": "2026-03-15", "value": 72, "unit": "bpm", "time": "08:00"}
  ],
  "sleep": [
    {"date": "2026-03-15", "duration": 7.5, "unit": "hours", "quality": "good"}
  ],
  "blood_sugar": [
    {"date": "2026-03-15", "value": 5.6, "unit": "mmol/L", "time": "fasting"}
  ]
}
```

## 使用方法

### 记录数据

- "记录体重 70.5kg"
- "记录血压 120/80"
- "记录心率 72"
- "记录睡眠 7.5小时，质量很好"
- "记录空腹血糖 5.6"

### 查询数据

- "查看最近一周的体重变化"
- "显示本月的血压记录"
- "分析我的睡眠趋势"

### 健康建议

- "我的健康指标正常吗？"
- "根据我的数据给出建议"

## 健康指标参考范围

| 指标 | 正常范围 | 备注 |
|------|---------|------|
| 体重 | BMI 18.5-24 | 根据身高计算 |
| 血压 | 90-140/60-90 mmHg | 静息状态 |
| 心率 | 60-100 bpm | 成人静息心率 |
| 空腹血糖 | 3.9-6.1 mmol/L | 空腹8小时以上 |
| 睡眠 | 7-9 小时 | 成人推荐时长 |

## 注意事项

1. 所有数据仅存储在本地，保护隐私
2. 健康建议仅供参考，不构成医疗诊断
3. 如有异常指标，请及时就医

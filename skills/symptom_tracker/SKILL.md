# 症状追踪技能

## 功能

- 记录身体症状（头痛、胃痛、疲劳等）
- 追踪症状严重程度和持续时间
- 关联可能触发因素
- 生成症状日志和分析

## 数据结构

症状记录存储在 `~/.nanobot/data/symptom_logs.json`:

```json
{
  "symptoms": [
    {
      "id": "sym_001",
      "date": "2026-03-15",
      "time": "14:30",
      "symptom": "头痛",
      "severity": "moderate",
      "severity_scale": 6,
      "duration": "2小时",
      "location": "前额",
      "triggers": ["长时间看屏幕", "睡眠不足"],
      "relief_measures": ["休息", "喝水"],
      "notes": "下午开始，休息一下后缓解"
    }
  ],
  "patterns": {
    "frequent_symptoms": ["头痛", "眼疲劳"],
    "common_triggers": ["睡眠不足", "久坐"]
  }
}
```

## 使用方法

### 记录症状

- "头痛，程度中等，在前额位置"
- "胃痛，可能是午餐引起的"
- "感觉很疲劳，持续了3天"
- "眼睛干涩，看屏幕太久了"

### 查询分析

- "最近有什么症状？"
- "分析我的头痛模式"
- "哪些因素经常触发不适？"

### 症状追踪

- "这周症状有改善吗？"
- "显示症状日历"
- "生成健康趋势报告"

## 症状严重程度分级

| 等级 | 描述 | 行动建议 |
|------|------|---------|
| 1-2 | 轻微，几乎不影响 | 观察，记录 |
| 3-4 | 轻度，略有不适 | 注意休息 |
| 5-6 | 中度，影响活动 | 采取措施，考虑就医 |
| 7-8 | 重度，严重影响 | 尽快就医 |
| 9-10 | 极重，无法忍受 | 立即就医 |

## 常见症状分类

### 头部
- 头痛、头晕、偏头痛

### 眼部
- 眼疲劳、视力模糊、干涩

### 消化系统
- 胃痛、恶心、腹泻、便秘

### 全身
- 疲劳、乏力、发烧、寒战

### 肌肉骨骼
- 背痛、颈痛、关节痛、肌肉酸痛

## 注意事项

⚠️ **以下情况请立即就医**：
- 胸痛或胸闷
- 呼吸困难
- 剧烈腹痛
- 持续高烧（>39°C 超过3天）
- 严重头痛伴呕吐
- 意识模糊或昏厥

## 免责声明

症状追踪仅用于个人健康管理参考，**不能替代专业医疗诊断**。如有健康问题，请及时咨询医生。

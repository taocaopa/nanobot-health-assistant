# 饮食追踪技能

## 功能

- 记录每日三餐和加餐
- 估算卡路里和营养成分
- 查询今日/历史饮食记录
- 生成饮食分析报告

## 数据结构

饮食记录存储在 `~/.nanobot/data/diet_logs.json`:

```json
{
  "2026-03-15": {
    "meals": [
      {
        "type": "breakfast",
        "time": "08:00",
        "items": ["燕麦粥", "鸡蛋", "牛奶"],
        "calories": 450,
        "notes": "吃得很饱"
      },
      {
        "type": "lunch", 
        "time": "12:30",
        "items": ["米饭", "鸡胸肉", "西兰花"],
        "calories": 650
      }
    ],
    "total_calories": 1800,
    "water_intake": 2000
  }
}
```

## 使用方法

### 记录饮食

- "早餐吃了燕麦、鸡蛋和牛奶"
- "午餐记录：米饭、红烧肉、青菜"
- "下午加餐：一个苹果"
- "喝水 500ml"

### 查询记录

- "今天吃了什么？"
- "显示本周饮食记录"
- "今天摄入多少卡路里？"

### 饮食分析

- "分析我的饮食习惯"
- "我的饮食健康吗？"
- "建议今天晚餐吃什么"

## 常见食物卡路里参考

| 食物 | 份量 | 卡路里 |
|------|------|--------|
| 米饭 | 1碗(150g) | 174 kcal |
| 鸡蛋 | 1个 | 70 kcal |
| 牛奶 | 250ml | 150 kcal |
| 苹果 | 1个(200g) | 100 kcal |
| 鸡胸肉 | 100g | 165 kcal |
| 燕麦 | 50g | 190 kcal |

## 建议每日摄入

- **成年男性**: 2000-2500 kcal
- **成年女性**: 1800-2200 kcal
- **水**: 2000-2500 ml

## 注意事项

1. 卡路里估算仅供参考，实际值可能有偏差
2. 建议多样化饮食，保证营养均衡
3. 特殊人群（孕妇、慢性病患者）需遵医嘱
